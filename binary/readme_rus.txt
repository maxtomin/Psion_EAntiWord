  Установка Antiword:
  1. Запустить StdLib.sis на настольном компьютере и следовать
     инструкциям на экране.
  2. После окончания установки скопировать Antiword.exe в любой
     каталог на Psion, например в "C:\Antiword"
  3. В ту же папку скопировать файл "cp1251.txt" из папки Resource
     и переименовать его в "encoding.txt"

  Ковертация вложений e-mail:
  1. Сохранить вложение на диск, лучше всего в папку "С:\Antiword"
  2. Запустить Antiword.exe, выбрав его и нажав на <Enter>.
  3. Ввести имя файла MS Word, в который сохранено вложение, и нажать Enter.
     Если файл находится в папке C:\Antiword, то можно ввести только
     его имя. В противном случае нужно ввести полный путь к файлу.
  4. Ввести имя выходного текстового файла (с расширением txt)
  5. Ввести максимальную ширину строки в выходном файле. Это нужно для
     автоматического переноса строк по словам.
     Обычно нужно сформировать файл без дополнительных переносов строк,
     в этом случае нужно просто нажать Enter.
  6. Если конвертация выполнена успешна, выводится сообщение
     "Converting complete". Полученный текстовый файл можно открыть в
     программах:
     а) EPOC Word (для этого выберите File - More - Import text file)
     б) EBook
     в) EPOC Text Editor
     и т.п.
